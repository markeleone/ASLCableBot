Welcome to the ASL CableBot documentation!
===================================

**CableBot** is a compliant robot test rig actuated by cables created by a team in Stanford's `Autonomous Systems Lab <https://stanfordasl.github.io/>`_

It was built to create a hardware platform to test *infinite dimensional* control software developed in the lab.


Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
